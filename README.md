# ChartSplit
```
usage: chartsplit.py [-h] --map MAP --outfile OUTFILE [--push] file_to_split

Split Master PDFs into parts

positional arguments:
  file_to_split      Master PDF to be split.

optional arguments:
  -h, --help         show this help message and exit
  --map MAP          path to yaml file with parts map
  --outfile OUTFILE  filename for output files
  --push             push files into Swing Shift folders
```
